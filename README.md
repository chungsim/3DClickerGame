# 3DClickerGame

###게임 소개
방치형 3D 게임 + 디펜스 <br>
8개의 플레이어 기물이 주사위를 굴리는 것으로 공격을 하여 밀려오는 적 기물을 막는 게임입니다.<br>

주사위는 장비와 같은 아이템으로 랜덤한 기물 눈(폰, 룩, 나이트, 비숍, 퀸, 킹) 6개를 가지고 있으며 눈은 중복 가능합니다.<br>
주사위는 상점에서 랜덤으로 생성할 수 있으며 1회에 100의 재화가 소모됩니다.<br>
생성 된 주사위는 인벤토리로 들어가며 인벤토리 창에서 주사위 선택 후 플레이어 기물에 대응하는 칸을 눌러 장착 가능합니다.<br>

적 기물을 처치하지 못한 상태로 플레이어 기물 칸에 도달하면 플레이어 체력이 1 감소합니다.<br>
체력이 0이 되면 사망하며 리스폰 대기 시간 이후 스테이지를 재개합니다.
